Privacy Policy

Last updated: July 4, 2026

ph-github-top displays public GitHub profile rankings for developers in the Philippines. It does not require login.

What we process

  • Public GitHub API data (usernames, follower counts, stars) cached for leaderboard display.
  • Anonymous usage in your browser (theme, metric preferences via URL params or local state).

What we do not do

  • We do not sell personal data.
  • We do not collect GitHub credentials.
